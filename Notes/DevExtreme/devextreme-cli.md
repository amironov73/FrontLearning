### devextreme-cli

Требования:

* Node.js v10.9.0 or later
* npm v6.2.0 or later

Как установить:

```
npm i -g devextreme-cli
```

Как создать приложение:

```
devextreme new vue-app app-name
```

Запустить:

```
npm run serve
```

Добавить вью в приложение:

```
devextreme add view view-name
```
